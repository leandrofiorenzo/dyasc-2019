# Diseño y arquitectura de sistemas de computación.

Trabajo práctico para la UNTREF

### Generar el jar.
+ mvn clean package
+ mvn package

### Ejecutar.
+ java -jar fibo.jar n

#### Ejemplos.
##### 1)
java -jar fibo.jar 5

resultado: fibo<5>: 0 1 1 2 3

##### 2)
java -jar fibo.jar 8

resultado: fibo<8>: 0 1 1 2 3 5 8 13
